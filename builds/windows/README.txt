Snake v1.0
==========

A classic grid-based Snake game built with Godot 4.

How to play
-----------

Launch Snake.exe. From the main menu, press Start Game.

Controls
--------

Move:    Arrow keys   or   W A S D
Pause:   Esc          or   P
Confirm: Enter        or   Space

Eat red apples to grow the snake and score points.
Avoid running into the walls or your own body.
Your high score is saved between sessions.

Save location
-------------

The high score file is written to your Windows user data directory:
    %APPDATA%\Godot\app_userdata\snaketaskmaster\highscore.save
